Projeto criação PDF - Boletim de Ocorencia

Fiz esse projeto visando agilizar a confecção dos BOs por parte do efetivo que esta trabalhando na atividade fim, com ele o militar já pode ir preenchendo
as informações e chegar na cidade da Policia com o PDF feito apenas para imprimir e assinar, não ficando dependente do computador coletivo lá presente.

Feito em HTML, CSS e JS.
Não grava dados nenhum pois não tem vinculo a BD e esta hospedado apenas no github utilizando do serviço de pages da própria plataforma.
